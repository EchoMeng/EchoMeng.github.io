伪类和伪元素

:link指示作为超链接（即有href）属性并指向一个未访问地址的所有锚。

:visited指示作为已访问地址超链接的所有锚。

:focus指示当前拥有输入焦点的元素，也就是可以接受键盘或者别的方式输入的元素。

:hover指示鼠标停留在的元素。

:active指示被用户输入激活的元素。例如鼠标指针停留在一个超链接上，如果用户点击鼠标，就会激活这个超链接。

:first-child指示第一个子元素。

:lang()根据语言选择。

:first-letter指示块级元素首字母。只能用于标记或者段落之类的块级元素，不能应用于超链接等行内元素。

first-line第一行文本。只能用于标记或者段落之类的块级元素，不能应用于超链接等行内元素。

:before，:after设置之前或者之后元素的样式。
