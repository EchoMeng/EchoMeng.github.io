记录MacBook必备的应用及文件以及安装方式。

终于结束了我的6.5年材料生涯，今天索性重装了系统，抹去磁盘，全部重来。所以这里备份一些需要用到的应用安装方式。

应用

  • Xcode
  • Chrome
  • 微信
  • vscode
    • python
    • debugger for chrome
  • shadowsocksX
  • QQ
  • QQMusic
  • Macdown

工具

  • homebrew
  • git
    git在Xcode里有安装，还需要配置一下用户名和邮箱：
  • python3
  • node.js, npm
  • hexo

技巧

  • 显示finder中隐藏文件：

defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder

  • 在finder顶部显示路径，并且可以邮件选择：

defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES
